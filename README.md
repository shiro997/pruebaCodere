# pruebaCodere
prueba técnica codere sesión práctica

Crear una aplicación ASP .NET en C# y páginas Web en Blazor que permita almacenar la información relativa a un grupo de empleados de una empresa de forma que:
1. Cada empleado tendrá la siguiente información: nombre, código entre [1,10000], puesto de trabajo, sueldo base, una referencia a su superior directo y una referencia al grupo al que pertenece.
2. Cada grupo tendrá información sobre: nombre de oficina, código numérico del área departamental entre [1 y 23] y una lista de referencias a sus empleados.

La aplicación deberá permitir editar, eliminar y consultar de los empleados.
Cree un servicio y vista que exponga la consulta de todos los empleados.
